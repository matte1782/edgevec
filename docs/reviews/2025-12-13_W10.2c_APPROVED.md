## Change Summary

**File:** `fuzz/fuzz_targets/graph_ops/target.rs`
**Line:** 100

```diff
- if let Ok(results) = index.search(vec, 10, &storage) {
+ if let Ok(results) = index.search(&vec, 10, &storage) {
```

**Explanation:** `storage.get_vector()` now returns `Cow<'_, [f32]>`. Adding `&` borrows from the Cow, which derefs to `&[f32]` as expected by `index.search()`.
